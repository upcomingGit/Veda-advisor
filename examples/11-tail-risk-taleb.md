# Example 11 - Taleb: Tail-Risk Filter Before a Leveraged Trade

## User question

Should I use leverage to increase returns on this position?

## What Taleb checks

1. Left-tail ruin risk.
2. Fragile versus robust payoff structure.
3. Whether a simpler non-levered path exists.

## Decision

Do not use leverage.

Potential tail damage is too large for this profile.

## Would be wrong if

1. Downside is strictly bounded and affordable under stress.
2. The user explicitly accepts a low-probability, high-impact loss in writing.
