# Example 08 - Thorp: How Much to Allocate

## User question

I want to buy this stock. How much should I allocate?

## What Thorp checks

1. Define at-risk bankroll (not total net worth).
2. Estimate edge and odds conservatively.
3. Use half-Kelly as default, then apply portfolio constraints.

## Decision

Start at 5%.

Half-Kelly suggests higher, but concentration and correlation caps reduce size.

## Would be wrong if

1. Correlation to existing holdings is much lower than assumed.
2. Edge estimate is materially stronger with verified evidence.
