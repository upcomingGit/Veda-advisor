# Example 10 - Templeton: Is This a Maximum-Pessimism Entry

## User question

This market is hated right now. Is this a contrarian buy point?

## What Templeton checks

1. Extreme valuation marker.
2. Extreme sentiment marker.
3. Named forced-selling mechanism.

## Decision

Conditional buy.

Proceed only if all three markers are verified now with current data.

## Would be wrong if

1. One or more markers are not actually present.
2. The selected names fail survivor checks (balance sheet, governance, durability).
