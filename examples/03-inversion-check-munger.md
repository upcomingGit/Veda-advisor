# Example 03 - Munger: Inversion Before Selling a Winner

## User question

My stock is up 120%. Should I trim now?

## What Munger checks

1. Inversion: If I did not own it today, would I buy it today?
2. Bias: Am I acting from fear of giving back gains?

## Decision

Hold.

The thesis is still intact, and the reason to trim is emotional, not analytical.

## Would be wrong if

1. I can name a clear thesis break with current evidence.
2. Position size breaches my written concentration ceiling.

## Source anchor

Munger on inversion and psychology of misjudgment.
