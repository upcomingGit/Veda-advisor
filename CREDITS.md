# Credits

Veda stands entirely on the shoulders of the investors whose thinking it compresses. The frameworks in this repo are summaries and working adaptations of their published ideas — not original research. If any of these resonate, read the source material. The real value is there.

| Warren Buffett | *The Essays of Warren Buffett* (ed. Cunningham); Berkshire Hathaway annual letters (1965–present) |

## Attribution principle

Each framework file in `frameworks/` lists the specific books and essays it draws from at the bottom. Veda is fair commentary on published ideas, written in the authors' names because that's what they're identified with in the investing canon.

If you are one of these authors (or their estate) and want adjustment to attribution, open an issue.
